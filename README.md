# ⚡ Metro-Power-Agent (捷運供電智慧維護代理系統 MVP)


這是一個基於 Generative AI (OpenAI GPT-4o-mini) 的供電設備故障診斷輔助工具原型 (PoC)。本系統模擬捷運供電廠的維修場景，協助工程師分析電纜絕緣與直流斷路器 (HSCB) 故障。

## ✨ 主要功能

- **AI 邏輯推理 (ReAct Pattern)**：模擬資深工程師的思考流程（分析 -> 查閱手冊 -> 匹配案例 -> 查詢規範 -> 診斷）。
- **自動化工程圖表生成**：
    - 📊 **PRPD (局部放電相位圖譜)**：針對 22kV 電纜 PD 告警，生成可視化相位分析圖。
    - 📈 **di/dt 特性曲線**：針對第三軌 HSCB 跳脫，生成電流上升率波形圖以判別短路或湧流。
- **直覺式互動介面**：使用 Streamlit 建構，支援側邊欄參數設定與 API Key 安全輸入。

## 🛠️ 安裝與執行

### 前置需求
- Python 3.8 或以上版本
- OpenAI API Key

📝 使用情境範例 (內建案例)
系統內建兩個經典故障案例供測試：

22kV 電纜絕緣異常：模擬 Qmax > 600pC 的內部空隙放電 (Void Pattern)。

HSCB 誤跳脫分析：模擬列車啟動湧流 (Inrush Current) 與短路故障的波形差異。
